# 三国文字Roguelike

文字冒险 + Roguelike + 三国题材

## 运行

```bash
cd /root/sanguo_roguelike
python3 engine.py
```

## 核心架构

```
config.py        # 配置（初始属性/事件时间表/阈值）
state.py         # 全局状态（时间/玩家/NPC/事件标记）
character.py     # 角色数据模型（武将/NPC/属性）
combat.py        # 叙事性战斗系统（预计算+生成）
narrative.py     # 文字生成（战斗/事件/死亡报告）
engine.py        # 主循环/事件调度
progression.py   # 解锁系统（永久进度）

events/
  mandatory.py   # 必然事件（184年黄巾~220年三国鼎立）
  conditional.py # 条件事件（桃园结义/三顾茅庐/随机遭遇）
```

## 游戏流程

1. **开局**（184年黄巾之乱）→ 玩家扮演散兵
2. **每月推进** → 触发事件（必然+条件）
3. **战斗** → 叙事性战斗，预计算胜负后生成文字
4. **死亡** → 史书记载 + 解锁新内容
5. **结局**（220年三国鼎立，或中途死亡）

## 解锁系统

死亡后可解锁：
- 新武将（可扮演）
- 传承技能
- 天赋
- 图鉴（人物资料）
- 史籍（背景故事）
